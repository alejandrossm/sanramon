# Alternativas UI

La paleta base se mantiene desde `docs/PALETA_UI.md`; estas alternativas cambian composicion, densidad, tipografia e iconografia sin proponer nuevos colores.

## Alternativa A: Operativa moderna

Estado actual de esta rama.

- Tipografia con stack moderno local: Segoe UI Variable, Segoe UI y system UI.
- Iconografia lineal local mediante sprite SVG reutilizable.
- Sidebar con iconos por modulo y acciones de tablas con icono + texto.
- Tarjetas, tablas y formularios con sombras mas suaves y jerarquia visual mas clara.

## Alternativa B: Institucional calida

- Mayor uso de acentos `Arena logo` y `Arena clara` en encabezados y paneles.
- Logo con mas presencia en login y dashboard.
- Titulos con mayor peso visual, manteniendo tablas densas.
- Iconos menos protagonistas, usados solo en navegacion y acciones primarias.

## Alternativa C: Administracion compacta

- Sidebar y tablas mas densas para pantallas de uso repetido.
- Acciones secundarias con iconos mas pequenos y texto reducido.
- Metricas en formato mas compacto, priorizando listados y filtros.
- Recomendable cuando crezcan los modulos de asistencia y reportes.
